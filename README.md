## How to run this project?
If you just downloaded this repository.
1) Open the terminal
2) Go into the map Portfolio
    ```shell
    cd .\portfolio
    ```
3) Install npm
    ```shell
    npm install
    ```
4) Install missing dependencies
5) And run the project
    ```shell
    npm run dev
    ```
